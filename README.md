# Portfolio Personnel

## Description
Ce portfolio est un site web moderne développé avec React et Vite qui présente mes compétences, expériences et projets en tant que développeur Full Stack. Il comprend mes réalisations professionnelles, mes compétences techniques, et mes certifications.

## Caractéristiques
- Design responsive avec Tailwind CSS
- Sections distinctes pour les expériences professionnelles, projets, et compétences techniques
- Présentation détaillée des projets avec captures d'écran
- Interface intuitive et navigation fluide
- CV téléchargeable
- Formulaire de contact

## Technologies utilisées
- React.js
- Vite
- Tailwind CSS
- JavaScript (ES6+)
- HTML5/CSS3
